# My Care Lab — 솔루션 탭 콘텐츠/구조 명세 (처음부터 디자인용)

이 문서는 **솔루션 탭**과 그로부터 이어지는 **에이전트 대화 → 플랜 생성까지의 모든 화면**의
**구조·내용·흐름**만 정리한 것입니다. **기존 화면/코드와 무관하게 처음부터 자유롭게 디자인**하세요.
(시각 스타일, 레이아웃, 색, 컴포넌트는 정해진 게 없습니다. 아래는 "무엇을 담아야 하는가"입니다.)

- 화면 규격: 모바일 1개 화면 (세로로 긴 폰 화면). 한 손 사용 전제.
- 톤앤매너: 다정하고 차분한 코치. 강요하지 않고 제안. 한국어. 이모지 가볍게 사용 가능.

---

## 0. 제품 한 줄

사용자가 **AI 에이전트와 대화하며 자기만의 건강 솔루션(체중감량)을 함께 설계**하고,
완성된 플랜을 홈에서 매일 넛징(작은 행동 제안)으로 이어가는 앱.

---

## 1. 전체 흐름 (화면 맵)

```
[솔루션 탭]
   │  "나만의 솔루션 만들기" 누름
   ▼
[에이전트 대화]  인트로 → Step1 현재상태 → Step2 목표체형 → Step3 타겟날짜 → Step4 플랜견적
   │  플랜 수락
   ▼
[솔루션 탭]  내 솔루션 카드 + 오늘의 넛징이 새로 생김
```

- 각 Step은 **에이전트 말풍선 → 사용자 입력 위젯 → 사용자 답변 말풍선** 순서로 쌓이는 대화형.
- Step2~4에는 **"← 이전" (이전 단계로)** 가능.

---

## 2. 솔루션 탭 (메인)

세로 스크롤 화면. 위에서 아래로:

### 2-1. 인사 영역
- 큰 인사: **"My Care Lab"**
- (선택) 짧은 서브 카피 한 줄

### 2-2. "기존 솔루션" 섹션
- 섹션 라벨: **기존 솔루션**
- 카드 2개 (더미, 이미 진행 중인 솔루션 예시):
  1. 🌿 **아침 5분 정원** — 태그 *Wellness Routine* — "하루를 깨우는 호흡과 스트레칭. 14일 챌린지로 운영됩니다." — 뱃지: 🧘 잘 다스리기 · D-14
  2. 😴 **밤 11시 잠들기** — 태그 *Sleep Track* — "취침 패턴을 정돈하고 회복 점수를 끌어올려요." — 뱃지: 😴 잘 자기 · D-30

### 2-3. "나의 솔루션" 섹션  *(플랜을 만든 뒤에만 등장)*
- 섹션 라벨: **나의 솔루션**
- 사용자가 방금 만든 솔루션 카드. 담을 내용:
  - 상태 리본: **진행 중**
  - 라벨: *My Solution*
  - 제목: **"{N}kg 가벼워지기"** (예: 5kg 가벼워지기)
  - 메타: "체지방률 {현재}% → {목표}% · 근육량 유지"
  - 4개 카테고리 아이콘 줄: 🔥 잘 태우기 / 😴 잘 자기 / 📉 잘 빠지기 / 🧘 잘 버티기
  - 카운트다운: Target = {이벤트명}, **D-{남은일수}**

### 2-4. "오늘의 넛징" 섹션  *(플랜을 만든 뒤에만 등장)*
- 헤더: **오늘의 넛징** + 우측에 **맥락 바꾸기 ↻** (시나리오 순환용, 점 인디케이터 4개)
- 넛징 카드 1장 (아래 4종 중 현재 1개 표시):
  - 구성: `⚠ {카테고리} 위협 감지` + 시각 / 제목 / 본문 / 버튼 **[✅ 했어요] [나중에]**
  - **4종 시나리오** (맥락 바꾸기로 순환):
    1. 🔥 **잘 태우기 위협 감지** (오후 5:42) — "최근 3일, 걸음수가 목표의 60%예요." / "오늘 퇴근길에 한 정거장 먼저 내려 걸어볼까요? 8,000보까지 약 1,200보 남았어요."
    2. 😴 **잘 자기 위협 감지** (밤 10:18) — "수면 점수가 2일째 60 이하예요." / "오늘은 30분만 일찍 침대로 가볼까요? 11시에 무드등을 자동으로 낮춰드릴게요."
    3. 📉 **잘 빠지기 위협 감지** (오전 9:05) — "주간 감량 페이스가 목표의 40%예요." / "오늘 저녁은 단백질 위주로 가볍게 어떠세요? 식단 사진 한 장으로 기록해두면 좋아요."
    4. 🧘 **잘 버티기 위협 감지** (오후 3:21) — "스트레스 지수가 2일째 70을 넘었어요." / "잠깐 멈춰서 5분 호흡 명상 어떠세요? 흔들리는 마음을 가라앉히는 데 도움이 돼요."
  - **[✅ 했어요]** → 보상 획득(별도 갤러리 화면에 반영). **[나중에]** → 카드 흐려지고 비활성.

### 2-5. "나만의 솔루션 만들기" 진입 카드  *(항상 표시)*
- 태그: *Co-Creation*
- 제목: **나만의 솔루션 만들기**
- 설명: "에이전트와 대화하며 나에게 꼭 맞는 플랜을 함께 만들어요."
- 큰 **＋** 어포던스. → 누르면 에이전트 대화 시작.

## 3. 에이전트 대화 (솔루션 만들기)

채팅형. 상단에 **진행 단계 표시** (예: "Step 1 / 4 · 현재 상태").
말풍선이 위로 쌓이고, 각 단계의 입력 위젯이 대화 흐름 안에 카드로 들어옴.

### 인트로
- 에이전트: **"안녕하세요. 어떤 솔루션을 함께 만들어볼까요?"**
- 사용자 선택지(빠른 응답): **[체중감량 솔루션 만들어줘] [수면 개선하고 싶어]**
- (데모에선 체중감량만 진행. 다른 선택 시: "이번 데모에서는 체중감량 솔루션만 시연돼요. 체중감량으로 진행할게요.")

### Step 1 / 4 · 현재 상태
- 에이전트: "좋아요. 먼저 현재 상태를 알려주세요. 전신사진과 체성분 데이터가 필요해요."
- 입력 위젯:
  - 📸 **사진 업로드** ("건너뛰면 더미 실루엣으로 진행돼요" — 선택)
  - 🧑 **신체 데이터**: 체중(kg) · 체지방률(%) · 근육량(kg) · 키(cm) · 나이(세) · 성별(F/M)
  - 📊 **최근 라이프스타일 평균**: 평균 걸음수(보) · 평균 수면(시간) · 수면 점수(/100) · 스트레스 지수(/100)
  - 버튼: **확인 완료**
  - (예시 기본값: 체중 65 / 체지방 32 / 근육 42 / 키 163 / 34세 / F / 걸음 4,200 / 수면 5.5h / 수면점수 58 / 스트레스 72)

### Step 2 / 4 · 목표 체형
- 에이전트: "이제 목표 체형을 설정해주세요. 슬라이더를 움직이면 신체 파라미터가 실시간으로 바뀌어요."
- 입력 위젯: 🎯 **목표 체형 슬라이더**
  - 좌측: **체형 미리보기** (현재 → 목표로 변형되는 실루엣/사진). 라벨 "현재" / "목표".
  - 우측: **실시간 수치 읽기** — 체중 / 체지방률 / 근육량 (슬라이더 따라 값과 증감 표시)
  - 슬라이더 눈금: "현재 체형" ↔ "목표 체형 →"
  - 버튼: **이 목표로 확정** (슬라이더를 움직여야 활성)
  - 이전 버튼 가능

### Step 3 / 4 · 타겟 날짜
- 에이전트: "📅 언제까지 달성하고 싶으세요? 메모리에서 다가오는 이벤트를 찾아봤어요."
- 입력 위젯: 🗓 **타겟 선택** — 목록에서 1개 선택:
  - **메모리 이벤트** (MEM 태그): 💍 My Wedding (2026-08-15) / 🌺 Trip to Hawaii (2026-10-03)
  - **프리셋**: 📆 3개월 후 / 6개월 후 / 1년 후
  - 각 항목에 날짜 + **D-{일수}** 표시
  - 버튼: **다음** (선택해야 활성)
  - 이전 버튼 가능

### Step 4 / 4 · 플랜 견적
- 에이전트: "데이터를 분석해서 플랜을 만들었어요. 견적을 확인해보세요."
- **플랜 견적 카드** — 담을 내용:
  - 타이틀: "체중감량 플랜 견적"
  - 큰 목표: **"{N}kg 가벼워지기"**
  - 요약 줄: 기간 **{일수}일** · 체지방률 **{현재}% → {목표}%** · 일일 적자 **{kcal}kcal**
  - **4개 카테고리 목표** (각 줄: 아이콘 · 라벨 · 세부 · 대표값). 일부는 **병목** 뱃지:
    - 🔥 **잘 태우기** — "매일 +{kcal}kcal · {목표 걸음수}보 (현재 {현재 걸음수})"
    - 😴 **잘 자기** — "{목표}시간 · 점수 {목표}↑ (현재 {현재}h/{현재점수})"
    - 📉 **잘 빠지기** — "주 -{kg}kg · 체중계 연동 권장"
    - 🧘 **잘 버티기** — "스트레스 {상한}↓ (현재 {현재})"
  - 중간점검 안내: "중간점검 D-{a}, D-{b} · 체중계 연동 권장"
  - **안전 경고** (조건부): "⚠ 일일 적자 {kcal}kcal는 권장 한도(1,000kcal)를 초과해요. 목표를 낮추거나 기간을 늘려주세요."
  - 버튼: **[✅ 이 플랜으로 시작] [⬆ 목표 올리기] [⬇ 목표 낮추기]** (안전 경고 시 '목표 올리기' 비활성)
  - 이전 단계로 링크 가능

### 완료
- 에이전트: "플랜이 저장됐어요. 홈에서 매일 함께 지켜볼게요. 🌱"
- → 솔루션 탭으로 복귀. 이제 "나의 솔루션" 카드 + "오늘의 넛징"이 보임.

## 4. 4개 카테고리 (의미 — 카피·아이콘 참고)

| 카테고리 | 핵심 질문 |
|---------|----------|
| 🔥 잘 태우기 | 오늘 충분히 움직였나? (활동·칼로리 소모) |
| 😴 잘 자기 | 회복이 됐나? (수면 시간·점수) |
| 📉 잘 빠지기 | 실제로 변하고 있나? (감량 페이스) |
| 🧘 잘 버티기 | 무너지지 않고 있나? (스트레스) |

- **"병목" 뱃지**: 현재 데이터가 가장 취약한 카테고리에 붙어 "여기부터 집중" 신호.

---

## 5. 플랜 수치가 의미하는 것 (디자인 시 어떤 데이터를 보여줄지 참고)

- 목표 감량 = 현재 체중 − 목표 체중
- 일일 칼로리 적자 = (감량 kg × 7700) ÷ 기간(일) → 1,000kcal 초과 시 **위험 경고**
- 잘 태우기: 좌식이면 더 많은 분담 → 걸음 목표 상향
- 잘 자기: 수면 부족/저점수면 목표치 상향(병목)
- 잘 빠지기: 주간 감량 = 총감량 ÷ 주수
- 잘 버티기: 스트레스 높으면 상한 더 낮게(병목)
- 중간점검: 기간의 1/3, 2/3 지점

> 디자이너 메모: 위 수치는 입력값에 따라 **동적으로 바뀜**. 카드는 "값 + 단위 + (현재→목표)" 패턴이 반복됨.

## 6. 디자인 자유도 / 부탁

- 위 **구조와 내용·카피**만 지켜주세요. 레이아웃·컴포넌트·비주얼은 처음부터 자유.
- 채팅 단계는 "대화가 위로 쌓이는" 흐름이면 되고, 입력 위젯의 형태(카드/시트/인라인 등)는 자유.
- 모바일 한 화면 기준, 한 손 조작·엄지 도달 영역 고려.
- 4개 카테고리(🔥😴📉🧘)는 앱 전체를 관통하는 축이므로 일관된 시각 언어로.
